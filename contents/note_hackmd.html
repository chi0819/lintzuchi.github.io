Neovim/Vim

	Why I use Neovim as my editor :

	Vim is an old editor, differ from modern editor like VScode, there are a lots of skills you should learn 
	before use Vim. A story about Vim is a game about how to escape Vim
	For Windows or MacOS user, it's akward to use h, j, k, l key to move your cursor on the screen. But if you 
	familiar with the operation of Vim, it's actually a powerful editor for developing.
	In my opinion, there is an upgrade version of Vim called Neovim. Compare Neovim with Vim, Neovim is more flexible
	than Vim, there are a lots of plugs on github support Neovim, so that you can decorate your workspace and add
	useful tools make coding more convenient and easily.
